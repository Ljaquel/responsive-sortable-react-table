# responsive-sortable-react-table
 A reusable table component made for react which is both responsive and sortable
# Installation
```
    npm install responsive-sortable-react-table -save
```
# Usage
```
    <Table/>
```
# Options
  content: Array of Objects | Required
    f
  headers: Object
  className: String
  breakpoint: Number
  onSort: Function
  sortDir: Number 1 | -1
  sortKey: String
  headerClass: String
  bodyClass: String
